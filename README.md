# Classification-Analysis

# Day 01 - Logistic Regression

## 📌 Overview
This project demonstrates the implementation of **Logistic Regression**, a supervised machine learning algorithm used for **classification problems**. The model is trained using the **Social_Network_Ads.csv** dataset to predict whether a user purchases a product based on certain features.

## 📂 Files Included
- `LogisticRegression.py` → Python script containing model training and prediction logic.
- `Social_Network_Ads.csv` → Dataset used for training and testing the Logistic Regression model.

## 🚀 What I Learned
- Loaded and explored a dataset using **Pandas**.
- Performed **data preprocessing** by selecting features and labels.
- Split the dataset into **training and testing sets**.
- Applied **feature scaling** using `StandardScaler` for better model performance.
- Trained a **Logistic Regression** model using `sklearn`.
- Predicted outputs on test data.
- Evaluated model performance using a **confusion matrix**.
- Visualized classification results for better understanding.

  
# Day 02 - SVM Classifier

## 📌 **Overview**  
This project demonstrates the implementation of the **Support Vector Machine (SVM) Classifier**, a supervised machine learning algorithm used for classification tasks. The model is trained using the same **Social_Network_Ads.csv** dataset to predict whether a user purchases a product based on certain features. SVM works by finding the optimal decision boundary (**hyperplane**) that best separates different classes in the dataset.

## 📂 **Files Included**  
- **SVMClassifier.py** → Python script containing model training, prediction, and visualization logic using the SVM classifier.  
- **Social_Network_Ads.csv** → Dataset used for training and testing the SVM model.  

## 🚀 **What I Learned**

- Performed data preprocessing by selecting features and labels.  
- Split the dataset into **training and testing sets**.  
- Applied **feature scaling using StandardScaler** to improve model performance.  
- Trained an **SVM Classifier using sklearn**.  
- Understood how SVM finds an **optimal hyperplane** to classify data points.  
- Predicted outputs on test data.  
- Evaluated model performance using a **confusion matrix**.  
- Visualized classification results to better understand decision boundaries.

# Day 03 - KNN Classifier

## 📌 **Overview**  
This project demonstrates the implementation of the K-Nearest Neighbors (KNN) Classifier, a supervised machine learning algorithm used for classification tasks. The model is trained using the same Social_Network_Ads.csv dataset to predict whether a user purchases a product based on certain features. KNN works by identifying the ‘K’ nearest data points to a new input and classifying it based on the majority class among its neighbors.

## 📂 **Files Included**
kNN.py → Python script containing model training, prediction, and visualization logic using the KNN classifier.  
Social_Network_Ads.csv → Dataset used for training and testing the KNN model.

## 🚀 **What I Learned**  
- Loaded and explored a dataset using Pandas.  
- Performed data preprocessing by selecting features and labels.  
- Split the dataset into training and testing sets.  
- Applied feature scaling using StandardScaler to normalize feature values for better distance calculation.  
- Trained a K-Nearest Neighbors (KNN) Classifier using sklearn.  
- Understood how KNN classifies data points based on the nearest neighbors.  
- Learned the importance of choosing an appropriate value of K.  
- Predicted outputs on test data.  
- Evaluated model performance using a confusion matrix.  
- Visualized classification results to better understand decision boundaries.
